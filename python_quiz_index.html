Python alapok — 10. osztály

45 kérdéses vegyes kvíz

Feleletválasztó, behelyettesítéses és programíró feladatok — előre megírt automatikus javítással a feleletválasztókhoz.

I. Feleletválasztós kérdések (20)

II. Behelyettesítés / Kódkiegészítés (15)

Ezeket a válaszokat a tanár kézzel is ellenőrizheti — a rendszer csak összegyűjti a beírt válaszokat.

III. Rövid programíró feladatok (10)

Ide írhatják a diákok a megoldásokat — ezek automatikusan nem lesznek javítva, a tanár ellenőrzi őket.

Mit csinál ez az oldal?

  • A 20 feleletválasztós kérdést azonnal javítja a rendszer.
  • A 15 behelyettesítéses és 10 programíró feladatot a tanár kézzel nézi át.
  • A válaszokat JSON-ként le lehet tölteni, hogy könnyen importálható legyen Excelbe vagy a tanári rendszerbe.
Rövid telepítési útmutató (ingyenes URL-hez)
  1. Hozz létre egy új GitHub repository-t. (pl. python-quiz)
  2. Helyezd el ezt az index.html fájlt a repo gyökerébe és pushold fel.
  3. GitHub Pages: Settings > Pages > Branch: main > / (root) > Save. A site a https://felhasználónév.github.io/repo címen lesz elérhető.
  4. Alternatív: Netlify Drop (drag & drop az index.html), vagy Vercel — mindegyik ingyenes kezdőknek.

Használat: A diák kitölti, rákattint az Ellenőrzés-re, majd a tanár letölti a JSON-t és manuálisan kijavítja a kódos feladatokat.
